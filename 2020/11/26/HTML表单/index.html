常见的表单的标签

标签 作用
<form> 定义表单
<input> 输入区域
<select> 选项组
<option> 下拉列表中的选项在<select> 选项组里面
<textarea> 文本域
<legend> 域标题

<input>标签常见的属性

常见的属性

常见的属性 作用
type 创建单行文本输入框
name 当前input起个名字，用作表单参数
value 定义input元素的默认值
maxlength 规定input元素可输入最长字符数
width 下拉列表中的选项在<select> 选项组里面
width 宽度
height 高度

<input>标签的type属性

应为type属性很重要我下面就介绍一下type属性

type值 作用
button 按钮
submit 提交按钮
text 输入框
file 文件上传
password 密码输入框
hidden 定义隐藏的输入字段
image 定义图像形式的提交按钮
checkbox 定义复选框
radio 单选框
reset 重置
tel 手机号码
email 邮箱地址
number 数字
date 日期
